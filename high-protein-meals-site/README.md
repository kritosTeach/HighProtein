## 📱 Preview Locally
1. **VSCode Live Server**: Right-click `index.html` → Open with Live Server
2. **Command line** (recommended):
   ```
   npx serve high-protein-meals-site
   ```
   Opens http://localhost:3000

3. **Python** (if installed):
   ```
   cd high-protein-meals-site
   python -m http.server 8000
   ```

## 🌐 Deploy (Free/Fast)
1. **Netlify**: Drag `high-protein-meals-site` folder to netlify.com/drop
2. **Vercel**: `npx vercel` (git required)
3. **GitHub Pages**: Push to GitHub repo, enable Pages
4. **Any hosting**: Upload folder (static HTML)

## 💰 Monetization Setup
1. **Google AdSense**: Replace `.adsense-placeholder` divs with your ad code
2. **Amazon Associates**: Update affiliate links in boxes
3. **Newsletter**: Integrate EmailJS/Formspree in JS forms
4. **GA4**: Uncomment script in HTML heads, add your ID
